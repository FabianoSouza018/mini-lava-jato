# 🚗 Sistema de Lava Jato (Projeto em C + HTML)

## 📌 Descrição
Sistema simples para gerenciamento de uma microempresa de lavagem automotiva, desenvolvido com foco em aprendizado prático de lógica em C e estrutura web com HTML.

## 🎯 Objetivo
- Praticar lógica de programação em C  
- Trabalhar com entrada e saída de dados  
- Simular um sistema real  

## 🧠 Funcionalidades
- Cadastro de cliente
- Escolha de serviços
- Cálculo de valores
- Promoção com desconto

## 💰 Serviços

- Lavagem Simples: R$ 60.00  
- Lavagem Completa: R$ 180.00  
- Polimento: R$ 320.00  
- Promoção: 20% de desconto  

## ⚙️ Tecnologias
- C
- HTML
- Linux (Ubuntu VM)
- Apache (futuro)
